🔄 4swap — Join the DAO in 3 Steps 🖊️ BscScan Write Contract

Contract (BSC): DAOOnboardingSwap 111111 — 0x0fAA9…0Dd1

What you get: On success, the contract sends you a fixed reward of 7,454,385,000 NWR5 automatically.

✅ What you need

  • 👛 A wallet on BSC mainnet (BNB Chain).
  • ⛽ A little BNB for gas + the swap amount (USD value per phase is built-in).

1️⃣ Open, Connect, Select

  1. 🔗 Open the button above (BscScan “Write Contract”).
  2. 🔌 Click “Connect to Web3”.
  3. 🧭 Find the swap() function. It’s payable (sends BNB).

2️⃣ How much BNB to send?

💡 The contract looks up the live BNB/USD price (Chainlink) and compares it to the current phase’s USD price. Just send enough BNB for the phase; extra is refunded automatically.

Check current price (optional):

  • 👓 In the Read Contract tab, call getCurrentPhase() — it returns (phase, usdPrice).
  • 📉 You can also see remainingSpots() and totalSwaps.

Tip: If you’re unsure, send a tiny bit more BNB — any excess is refunded within the same transaction.

🧭 How to Use the Level Endorsement Contract 🖊️ BscScan Write Contract

Endorsing levels is simple: bring your NWR5, follow the steps, and the contract will handle price checks, rewards, badges, and logs on-chain.

🔗 Contract & Token Links

BSC

NWRLevelContributionVault (BSC): Read • Write NWR5 (BSC): Read • Write

Polygon

NWRLevelContributionVault (Polygon): Read • Write NWR5 (Polygon): Read • Write

🧱 Prerequisites

  • 🔐 A wallet connected to BSC or Polygon.
  • 🪙 Enough NWR5 to cover the level price (see below).
  • ✅ Approve the vault contract to spend your NWR5 before endorsing:
    Use your NWR5 token’s approve(spender, amount) in the explorer “Write” tab.
    Spender = vault address above • Amount = price for the level you’re endorsing.

💵 Level Prices (contract-enforced)

User Levels (1 → 11): 10,000 → 25,000 → 50,000 → 100,000 → 200,000 → 300,000 → 450,000 → 600,000 → 750,000 → 900,000 → 1,000,000 NWR5

DAO Levels (1 → 11): 100,000,000 → 200,000,000 → 300,000,000 → 400,000,000 → 500,000,000 → 600,000,000 → 700,000,000 → 800,000,000 → 900,000,000 → 950,000,000 → 1,000,000,000 NWR5

💡 The contract returns 50% of the price back to the endorser as a reward—automatically.

👤 Endorse a User Level

  1. 👁️ Check current level: userLevel(address) (Read tab).
  2. 🎯 Choose the next level (must be current + 1 and ≤ 11). You can’t skip.
  3. 🪙 Approve NWR5 spending: approve(vault, userPrice[level]) on the NWR5 token.
  4. 🖊️ Call vault: endorseUserLevel(user, level) (Write tab).
  5. 🎁 You instantly receive 50% of the price back; the user levels up; badge may mint; logs emit on-chain.

Rules the contract enforces: • You can’t endorse above your own user level. • Levels must be sequential. • Payment is in NWR5 only; transferFrom must succeed.

🏛️ Endorse a DAO Level

  1. 🧱 Check that the user is already user level 11.
  2. 👁️ Check user’s daoLevel(address).
  3. 🎯 Choose the next DAO level (must be current + 1 and ≤ 11).
  4. 🪙 Approve NWR5 spending: approve(vault, daoPrice[level]) on the NWR5 token.
  5. 🖊️ Call vault: endorseDaoLevel(user, level) (Write tab).
  6. 🎁 You instantly receive 50% back; user advances DAO level; optional vote eligibility is notified; logs emit.

Rules the contract enforces: • You can’t endorse above your own DAO level. • User must be User L11 before any DAO level. • Sequential only; payment in NWR5 must clear.

🧩 Function Signatures

endorseUserLevel(address user, uint8 level) endorseDaoLevel(address user, uint8 level)

Helpful reads: userLevel(user), daoLevel(user), getUserStats(user)
